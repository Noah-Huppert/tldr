# git

> Distributed version control system.

- Show file changes:

`git status`

- Track file changes:

`git add {{file}}`

- Commit file changes:

`git commit -m "{{Message}}"`

- Upload changes to remote server:

`git push {{server name}} {{branch}}`

- Switch to a branch:

`git checkout {{branch}}`

- Create a new branch:

`git checkout -b {{branch}}`
